## METHODOLOGY

### Evidence-Grade Standards

All deliverables follow strict accountability standards:

1. **No Speculation**
   - Every claim backed by verifiable source
   - No assumptions or inferences presented as fact

2. **Strict Citations**
   - Document name + date + page/section/line number
   - Verification status disclosed (verified_primary, verified_secondary, needs_verification)

3. **Contradiction Mapping**
   - Identify conflicts between sources
   - Cite both sides of contradiction
   - Explain implications

4. **Accountability Assignment**
   - Legal exposure (statutes violated, enforcement mechanisms)
   - Political accountability (electoral consequences)
   - Administrative responsibility (process failures, remedial actions)

5. **Traceability**
   - All claims traceable to primary sources
   - Evidence matrix serves as single source of truth
   - Cross-references maintained throughout
